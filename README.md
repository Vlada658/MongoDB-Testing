# MongoDB-Testing
**Описание задания:**  
Создана база данных и коллекция в MongoDB на основе данных из MySQL с последующим выполнением запросов.

**Что было сделано:**  
- Создан кластер в MongoDB Atlas  
- Создана база данных qa_shop и коллекция card_info  
- Перенесены данные из таблицы card_info (MySQL) в MongoDB (документы)  
- Выполнены запросы к коллекции с использованием:
  - фильтрации (по статусу, типу карты, дате)  
  - логических операторов (AND, диапазоны значений)  
  - поиска по строке  
  - агрегатных функций (подсчет количества)
